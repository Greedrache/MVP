<h2>📌 About the Project</h2>

<p>
This repository contains the source code of the official MVP website.
The platform serves as a central hub for information about the party and enables active participation.
</p>

<p>
It provides structured access to key political content such as the party statutes, program, and organizational transparency, while also offering ways for users to get involved.
</p>

<hr>

<h2>⚙️ Technologies</h2>

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
  <li>PHP (for selected server-side functionality)</li>
</ul>

<p>
The project intentionally avoids heavy frameworks in order to remain lightweight, maintainable, and easy to deploy.
</p>

<hr>

<h2>🚀 Features</h2>

<ul>
  <li>📜 Structured presentation of the party statutes</li>
  <li>🗳️ Public access to the political program</li>
  <li>🤝 Participation and application options</li>
  <li>📊 Transparency regarding democratic structures</li>
  <li>❓ FAQ section for common questions</li>
  <li>📄 Downloadable documents (e.g. PDFs)</li>
  <li>📱 Social media integration</li>
  <li>⚖️ Complete legal notice (Imprint)</li>
</ul>
