# 👑 MVP

**Participation. Responsibility. Perspective.**

Official website of MVP – a political project focused on direct participation, transparency, and strengthening local structures.

🌐 **Live at:** [mvp-politik.de](https://mvp-politik.de)

---

## 📌 About the Project

This repository contains the source code of the official MVP website.
The platform serves as a central hub for information about the party and enables active participation.

It provides structured access to key political content such as the party statutes, program, and organizational transparency, while also offering ways for users to get involved.

---

## ⚙️ Technologies

- HTML
- CSS
- JavaScript
- PHP (for selected server-side functionality)

The project intentionally avoids heavy frameworks in order to remain lightweight, maintainable, and easy to deploy.

---

## 🚀 Features

- 📜 Structured presentation of the party statutes
- 🗳️ Public access to the political program
- 🤝 Participation and application options
- 📊 Transparency regarding democratic structures
- ❓ FAQ section for common questions
- 📄 Downloadable documents (e.g. PDFs)
- 📱 Social media integration
- ⚖️ Complete legal notice (Imprint)
